# Stack

- Svelte
  - Javascript framework
- Vite
  - Run and build tool
- Histoire
  - Story libraries to explore components
- Vitest
  - Test library for Vite

# Install dependencies

    pnpm install

# How to run histoire (interface to explore components)

    pnpm run story:dev

# You can then (probably) see the components in

    http://localhost:6006
