# Stack

- Svelte
  - Javascript framework
- Vite
  - Run and build tool
- Storybook
  - Story library to explore components
- Vitest
  - Test library for Vite

# Install dependencies

    pnpm install

# How to run Storybook (interface to explore components)

    pnpm run storybook

# You can then see the components in

    http://localhost:6006
